Утилита выполняет слияние двух файлов относительно общего исходника.

Пример вызова:
merge <params> fileA fileB originalFile outputFile

params:
silent			- не выводить сообщения

При указании нескольких методов сравнения, будет выбран тот, который выше в этом списке (:
CryptoHash		- сравнивать крипто ключи вместо значений строк
CheckSum		- сравнивать контрольные суммы (по умолчанию)
String_equal	- сравнивать сроки посимвольно
