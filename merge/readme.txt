Утилита выполняет слияние двух файлов относительно общего исходника.

Примечание: При обработке файлов размером больше 1Мб, переключается
на алгоритм, не умеющий определять наибольшую общую последовательность.
Возможно, будет исправлено в следующих версиях.

Пример вызова:
merge <params> fileA fileB originalFile outputFule

params:
silent - не выводить сообщения
notIncludeOriginal - не указывать версию исходного файла в спорных блоках

Шаблоны сравнения (по умолчанию все используются)
Шаблон fileA_Original_fileB__result. 
N   - элемент отсутствует 
X,Y - значения
any - не важно
mrg - значения будут смержены повторно
что бы отключить шаблон, добавьте его в params

X_N_N__X
N_N_X__X
X_any_X__X
X_N_Y__mrg
X_X_N__N
N_X_X__N

// задел на будущее
При указании нескольких методов сравнения, будет выбран тот, который выше в этом списке (:
string.Equal()	- встроенный метода класса String (по умолчанию)
CryptoHash		- сравнивать крипто ключи вместо значений строк
CheckSum		- сравнивать контрольные суммы
